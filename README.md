# Employee Performance Analysis

## Project Overview:
- Employee Performance Analysis is a project developed using Python and SQL.
- The main objective of this project is to store employee data in a database and analyze their performance using Python.
- This project demonstrates how Python can be integrated with SQL to perform real-world data analysis tasks such as evaluating employee performance, identifying top performers and generating summary insights.


## Technologies Used:
- Python
- SQL (SQLITE)


## Key Features:
- Stores employee and performance data in database.
- Retrieves data using SQL Queries in SQLite.
- Analyzes performances metrics using Python.
- Identifies top-performing employees.


## Conclusion:
This project provides a practical approach to analyzing employee performance using Python and SQL. It demonstrates database integration, data analysis and performance evaluation - all useful skills for real-world applications.
